3D Visualization of Clustering of Cancer Samples
The expression data of over 350 samples of different cancer type are processed and Principle Component Analysis is performed to find out clusters.
The result is shown in an interactive 3D visualization which users could move around and zoom in/out, and reveal sample details by hovering. The samples could be filtered and colored by their attributes on the side bar.
Python scikit-learn Javascript d3.js three.js Data Analysis Data Visualization
View scripts on Github
Analysing World Climate Data from Observatory
Analysing temperature from observatory all over the world for the past 20 years.
Parallel processing with Spark was used to extract data, map them to the corresponding location, and calcuate the yearly average. Temperature data in void areas is also predicted.
Finally a color scale is applied and the data is visualised on a map.
Scala Spark Data Analysis Distributed System
View scripts on Github
Artwork Created by Neural Style Transfer
Neural Style Transfer uses a trained convolutional network and applies it on a different task, hence "transferring" what it has learnt.
The trained network has the ability to "recognize" the outline (higher level features) and patterns (correlation between channels of each layer).
By combining the outline of the original picture and the patterns of the style of another picture, a new "artwork" could be created.
Python Tensorflow Machine Learning Neural Network Data Visualization
View scripts on Github
